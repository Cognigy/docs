## Description

This Node executes either one or another part of a [Flow](/ai/agents/develop/projects-and-flows/overview) based on a condition. An If Node has two child Nodes: a _Then Node_ and an _Else Node_. The condition is evaluated during the execution of the Flow. If the condition evaluates to `true`, the Flow takes the path of the Then Node; if it evaluates to `false`, the path of the Else Node is executed.

## Parameters

### Condition Field
  
In the **Condition** field, use [CognigyScript](/ai/platform-features/cognigyscript) without `{{ }}`. For example, `input.slots.city` is all that is required. This evaluates to `true` if an Input Keyphrase with the tag `city` is found.
