## Description

The Enable Slot Fillers Node enables a particular Slot Filler. Use the drop-down menu to select the Slot Filler that you want to enable. For more information on Slot Fillers, refer to the [Natural Language Understanding (NLU)](/ai/platform-features/nlu/overview) page.
