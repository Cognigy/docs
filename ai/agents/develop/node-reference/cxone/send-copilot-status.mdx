## Description

The Send Status to CXone Copilot Node allows you to send a completion status to [NiCE CXone Copilot for Agents](https://help.nicecxone.com/content/agent/cxoneagent/enlightencopilotforagentscxa.htm). This Node allows you to provide human agents using the NiCE CXone Agent Workspace with information about the Flow status, indicating whether it succeeded or failed. For details, refer to the [NiCE CXone documentation](https://help.nicecxone.com/content/cognigyai/nativenodes/sendnode.htm#Add).

## Restrictions

- This Node is supported only in Cognigy.AI installations that are integrated with NiCE CXone.
