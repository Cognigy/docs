## Handover Providers

Endpoints that support handover have a **Handover Settings** section in their Endpoint settings.
In this section,
you can select and configure the handover provider to which the user will be forwarded when a [Handover to Human Agent Node](/ai/agents/develop/node-reference/service/handover-to-agent) is triggered.
Cognigy.AI offers support for the following handover providers:

- [Cognigy Live Agent](/ai/escalate/handover-reference/live-agent)
- [Chatwoot](/ai/escalate/handover-reference/chatwoot)
- [RingCentral Engage](/ai/escalate/handover-reference/ring-central-engage)
- [Salesforce](/ai/escalate/handover-reference/salesforce)
- [Genesys Cloud Guest Chat](/ai/escalate/handover-reference/genesys-cloud-guest-chat)
- [Genesys Cloud Open Messaging](/ai/escalate/handover-reference/genesys-cloud-open-messaging)
- [8x8](/ai/escalate/handover-reference/8x8)
