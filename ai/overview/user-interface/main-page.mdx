In Cognigy.AI's **Main** page, you can manage your Projects. 

The following image provides an overview of the **Main** page:

<Frame>
  <img class="image-center" src="/_assets/ai/overview/user-interface/main-page.png" style={{ width: 'auto' }} />
</Frame>

<AccordionGroup>
  <Accordion title="1. Project List">
    In the **Project List**, you can pin or unpin a Project by hovering over a Project, clicking the ![vertical-ellipsis](/_assets/icons/black-and-white/vertical-ellipsis.svg) button, and selecting **Pin Project** or **Unpin Project**.

    The ![plus](/_assets/icons/plus.svg) button opens the [Project Creation Wizard](/ai/agents/develop/projects-and-flows/projects).
  </Accordion>

  <Accordion title="2. Top Bar">
    |                       Button                       | Description                                                                                                                                              |
    |:--------------------------------------------------:|----------------------------------------------------------------------------------------------------------------------------------------------------------|
    |      ![help](/_assets/icons/blue/help.svg)      | Opens the Help Center search widget, where you can search for answers to potential questions in the [Cognigy Help Center](https://support.cognigy.com/). |
    | ![user-menu](/_assets/icons/blue/user-menu.svg) | Opens the [User Menu](/ai/administer/user-menu/overview).                                                                                              |
    
  </Accordion>

  <Accordion title="3. Welcome Dashboard">
    | Section     | Description                                                          |
    |-------------|----------------------------------------------------------------------|
    | News        | Includes Cognigy.AI updates.                                         |
    | Get Started | Links to the [Cognigy.AI Help Center](https://support.cognigy.com/). |
  </Accordion>
</AccordionGroup>
