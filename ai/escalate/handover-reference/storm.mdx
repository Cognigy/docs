## Description

The storm handover provider bridges Cognigy.AI and [storm](https://www.contentguru.com/en-gb/products/storm-cx/omni-channel-customer-engagement/omni-channel-agent-desktop/), the contact center platform from Content Guru, enabling end users to connect with human agents working in a contact center that uses storm.

## Prerequisites

- Access to [storm DTA](https://www.timeforstorm.eu/dta).

## Configuration on the Cognigy.AI Side


1. Go to **Build > Handover Providers** and click **+ New Handover Provider**.
2. In the **New Handover Provider** window, select **storm®** from the list and name the connector.
3. Configure the following fields:

    - **Base API URL** — the URL of your configured service in storm.
    - **Storm Authentication** — select a storm connection from the list. Alternatively, click ![plus-sign](/_assets/icons/black-and-white/plus-sign.svg) next to the field to add a storm connection. Fill in the following fields:

    | **Field**       | **Description**                                     |
    |-----------------|-----------------------------------------------------|
    | Connection name | Enter a unique name for your connection.            |
    | API Secret      | Enter the API secret for your Content Guru service. |
    | API Key         | Enter the API key for your Content Guru service.    |
