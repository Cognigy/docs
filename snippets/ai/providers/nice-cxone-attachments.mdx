<Accordion title="Attachments">
  When using a [Webchat v3 Endpoint](/webchat/v3/configuration) with the NiCE CXone handover provider, end users can send and receive file attachments during a handover session.

  End users can upload, share, and receive the following file types from Webchat v3 with human agents:

  | Category  | Supported formats                |
  |-----------|----------------------------------|
  | Documents | `.pdf`, `.docx`, `.pptx`, `.txt` |
  | Images    | `.jpeg`, `.jpg`, `.png`          |
  | Video     | `.mp4`, `.mov`                   |
  | Audio     | `.mp3`                           |
  | Links     | Web links/URLs                   |

  To activate attachment uploads from Webchat v3, configure [File Storage](/ai/agents/deploy/endpoints/file-storage) in the Webchat v3 Endpoint settings.
</Accordion>
