<Accordion title="Configure Attachments">
  When using a [Webchat v3](/webchat/v3/configuration) or [WhatsApp](/ai/agents/deploy/endpoint-reference/whatsapp) Endpoint with the NiCE CXone handover provider, end users can send and receive file attachments during a handover session.

  End users can upload, share, and receive the following file types:

  | Category  | Supported formats                            |
  |-----------|----------------------------------------------|
  | Documents | `.pdf`, `.docx`, `.pptx`, `.txt`             |
  | Images    | `.jpeg`, `.jpg`, `.png`                              |
  | Video     | `.mp4`, `.mov`                               |
  | Audio     | `.mp3`, `.ogg`              |
  | Links     | Web links/URLs                               |

  To activate attachment uploads, configure [File Storage](/ai/agents/deploy/endpoints/file-storage) in the Webchat v3 or WhatsApp Endpoint settings.
</Accordion>
