## Before you Begin

Check the Agent Copilot availability for your installation:

- For trial and shared SaaS, Agent Copilot is automatically activated.
- For dedicated SaaS, contact [Cognigy technical support](https://docs.cognigy.com/help/get-help/).
- For on-premises, install Agent Copilot with [Cognigy.AI Helm Chart](https://github.com/Cognigy/cognigy-ai-helm-chart).
