<Warning>
  Upcoming removals:

  - The `POST /v2.0/optionsresolver` API method will be removed in release 2026.3 for shared SaaS installations. If you use this method, update your setup before the removal to avoid disruptions.
  - Anthropic's `claude-3-opus` and Amazon Bedrock's `amazon.titan-text-express-v1` models will be removed in release 2026.3. Switch to the newer models before the removal.
  - Following the deprecation of Webchat v2, the full removal of this product is scheduled for February 2026. Make sure you migrate to [Webchat v3](/webchat/migration) before the removal date.

</Warning>
