Cognigy Webchat v2 is compatible with the browsers listed in the table.

| Browser         | Version            |
|-----------------|--------------------|
| Google Chrome   | 66 or higher       |
| Mozilla Firefox | 57 or higher       |
| Microsoft Edge  | 16 or higher       |
| Safari          | 11.1 or higher<sup>[1](#footnote1)</sup> |

<WebchatRestrictions/>

---

<sup id="footnote1">1</sup>: Cognigy only supports Websockets on Safari, which means that the fallback to long-polling is not allowed.
