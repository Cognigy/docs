The Cognigy Voice Gateway Self-Service Portal is a web-based interface that allows users to manage their accounts, settings, and preferences for Cognigy Voice Gateway.

The Web App has the following entities:

- [Users](/voice-gateway/webapp/users)
- [Accounts](/voice-gateway/webapp/accounts)
- [Applications](/voice-gateway/webapp/applications)
- [Recent Calls](/voice-gateway/webapp/recent-calls)
- [Alerts](/voice-gateway/webapp/alerts)
- [Carriers](/voice-gateway/webapp/carriers)
- [Speech](/voice-gateway/webapp/speech-services)
- [Phone Numbers](/voice-gateway/webapp/phone-numbers)
- [Outbound Call Routing](/voice-gateway/webapp/outbound-call-routing)


## How to start

If you use the Cognigy Voice Gateway Self-Service Portal for the first time, ensure that you have completed the steps in the [Getting Started guide](/voice-gateway/getting-started). Once you have completed the guide, proceed with the following steps:

1. Identify the [user scope](/voice-gateway/webapp/users) you have or add a new user. The operations that you can perform in Voice Gateway are based on your user scope, such as Admin, Service Provider, or Account.
2. Create an [account](/voice-gateway/webapp/accounts) or use an existing one.
3. Create an [application](/voice-gateway/webapp/applications) or use an existing one.
4. Add a [new carrier](/voice-gateway/webapp/carriers) or use an existing one.
5. Add a [phone number](/voice-gateway/webapp/phone-numbers) or use an existing one.
6. Add [speech provider credentials](/voice-gateway/webapp/speech-services) or use existing ones.

Additionally, you can:

- Check [recent calls](/voice-gateway/webapp/recent-calls) and view all information about the call.
- Check [alerts](/voice-gateway/webapp/alerts) if something went wrong with call configuration.
- Configure [outbound call routing](/voice-gateway/webapp/outbound-call-routing).
